R-Risk-Assessment Demo

Bem-vindo à demonstração do projeto R-Risk-Assessment de Gabriel Demetrios Lafis. Este projeto oferece ferramentas para avaliação e gerenciamento de riscos utilizando a linguagem de programação R.

Abaixo, você pode ver um exemplo de matriz de risco gerada pelo script main.R. Esta visualização ajuda a identificar e priorizar riscos com base em sua probabilidade e impacto.
